# MiniLove - 情感支持社区

## 项目简介
MiniLove 是一个面向18-45岁男性的图文+H5应用，提供情感支持、目标引导和社交连接。用户在晚间时段（18:00~04:00）使用，缓解缺乏目标、孤独感等情绪问题。

## 技术架构
- **前端**: Vue 3 + Vite + Tailwind CSS + PWA
- **后端**: Node.js + Express + PostgreSQL
- **部署**: 云服务器 + Docker + Nginx

## 项目结构
```
MiniLove-project/
├── frontend/          # 前端代码
├── backend/           # 后端代码  
├── database/          # 数据库设计
├── docs/              # 项目文档
├── scripts/           # 部署脚本
└── README.md          # 项目说明
```

## 核心功能
1. **情感社区** - 图文分享、话题讨论
2. **情感支持** - 树洞倾诉、共鸣匹配
3. **会员服务** - 个性化目标规划、专属内容
4. **社交互动** - 聊天室、同城活动
